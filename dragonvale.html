DragonVale World
Overview
DragonVale World was a free-to-play mobile game and a followup to its wildly successful predecessor, DragonVale.

We took the ideas and designs of the original DragonVale and further improved upon the overall user experience, onboarding, monetization, and graphical fidelity.
Role
Game design, UX design, Interaction design, User research, User flow, A-B Testing

I was also the lead designer for the first global event for the proudct, the "Exploration Extravaganza."
Core Loop
The Glow Bugs wireframes were designed to reuse as many different hud and ui elements as possible to keep filesize low and loading fast. Also, any area which was able to generate graphics in code was taken to keep the filesize under 50mb so the game could be downloaded without requiring wifi.
